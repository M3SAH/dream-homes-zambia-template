# Zambia Property Listings Template

A clean, fast, mobile-first **single-page real estate template** designed specifically for the Zambian market.

Showcase houses, flats, townhouses, plots/stands and small holdings for **sale** and **rent** in Lusaka, Kitwe, Ndola, Livingstone, Chongwe, Solwezi and other areas — all with ZMW pricing.

## Features

- Fully responsive & mobile-first design
- Filter properties by **location**, **type (sale/rent)** and **max price (ZMW)**
- Clickable property cards → detailed modal view
- Client-side contact form (demo alert – ready for Formspree/Netlify Forms integration)
- Easy to customize — no frameworks, no build step, no backend
- Zambian-focused: realistic locations, ZMW currency formatting, local-style placeholder properties

## Technologies

- HTML5
- CSS3 (custom properties/variables)
- Pure Vanilla JavaScript (no dependencies)

## How to Use

1. **Download** or clone this repository
2. Open `index.html` in any modern browser → works offline
3. **Customize**:
   - Change text, logo, colors → edit `index.html` and `assets/css/style.css.`
   - Add/edit/remove properties → edit the `properties` array in `assets/js/script.j.s`
   - Replace images → use Unsplash links or place your own photos in `assets/image.s/.`

### Quick Customization Examples

**Change site name & hero text**  
→ Open `index.html` → edit lines near `<title>` and `<h1>`

**Add a new property**  
→ Open `assets/js/script.js` → copy-paste one object in the `properties = [ ... ]` array and modify values

**Change main color**  
→ Open `assets/css/style.css` → edit `--primary-color` in `:root`
